WebGL 2.0 Fluid Convection Simulation

I added a nice system to view the simulation and some simple user input. The simulation itself hasn't changed much.

Choose a resolution. The higher the resolution, the slower it will run!

Horizontal resolution 3000
Vertical resolution 300

Controls

  • Middle mouse button to drag the simulation window arround.
  • Mouse wheel to zoom in and out (if you don't have a middle mouse button you can also use this to change the position of the simulation window.
  • Left mouse button to add heat. Hold down ctrl (command on mac) to remove heat.
  • Unfortunately touchscreen input is not supported yet.
  • Click the fullscreen button!
