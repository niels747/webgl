Controls

Mouse

  • Middle mouse button to drag the simulation area arround.
  • Mouse wheel to zoom in and out, Or to change brush size while holding down B
  • Left mouse button to use selected tool. Hold down Ctrl (command on mac) to do the opposite.
  • Touchscreen input is not supported yet.

    • Keys

    Number Keys         Select Display Mode

    G Show / Hide Sounding Graph
    H Show / Hide GUI
